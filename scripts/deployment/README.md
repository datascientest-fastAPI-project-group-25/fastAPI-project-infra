# Deployment Scripts

This directory contains scripts for deploying the infrastructure.

## Scripts

- `deploy.sh`: Main deployment script
- `deploy-infra.sh`: Deploy infrastructure components
- `deploy-targeted.sh`: Deploy specific infrastructure components
- `deploy-with-oidc.sh`: Deploy infrastructure with OIDC authentication

## Usage

These scripts should be run from the root of the repository:

```bash
./scripts/deployment/deploy.sh
```
